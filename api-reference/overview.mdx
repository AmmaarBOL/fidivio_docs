<Note>
  API documentation is coming soon. Complete OpenAPI specification and interactive endpoint documentation will be available here.

## Base URL

```text
https://api.fidivio.com/v1
```

**Sandbox:**

```text
https://sandbox-api.fidivio.com/v1
```

## Authentication

```bash
curl https://api.fidivio.com/v1/members \
  -H "Authorization: Bearer fidivio_live_YOUR_KEY"
```

See the [API Integration Guide](/technical/api-guide) for full authentication details.
