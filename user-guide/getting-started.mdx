# Getting Started

## 1.1 Logging In

### First-Time Sign In (Setup Required)

1. Go to the Fidivio login page in your web browser.
2. Enter your registered email address and password, then click **Login**.
3. You will be prompted to set up your authenticator app for two-factor authentication (2FA).
4. Scan the QR code using your authenticator app (e.g. Google Authenticator, Microsoft Authenticator).
5. Enter the 6-digit verification code displayed in your app to complete setup.

### Subsequent Sign Ins

1. Enter your email and password, then click **Login**.
2. Enter the 6-digit code from your authenticator app when prompted.

<Info>
If you forget your password, click **Forgot Password** on the login screen. A reset link will be sent to your registered email address within a few minutes. If you have lost access to your 2FA device, contact your Administrator to reset your 2FA.
</Info>

## 1.2 Initial Setup Checklist (Admin)

Complete these steps in order before using the platform:

1. **Set up the Program:** Setup & Support → Configuration → Program Setup
2. **Set up Partners:** Setup & Support → Partners
3. **Set up Users and Roles:** Setup & Support → Configuration → User Management and Role Management
4. **Input historical or current program data** via Data Input
5. **Review the Dashboards** — all key metrics should begin populating
