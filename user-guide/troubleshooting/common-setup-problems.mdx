## Dashboards show no data

<AccordionGroup>
  <Accordion title="Data not uploaded">
    Confirm data exists under **Data Input** for the selected period. Check the **Batches** tab for successful uploads.
  </Accordion>
  <Accordion title="Program not configured">
    Complete [Initial setup checklist](/user-guide/getting-started/initial-setup-checklist) — program setup, partners, and roles.
  </Accordion>
  <Accordion title="Wrong date range">
    Verify dashboard date filters cover periods with uploaded data.
  </Accordion>
</AccordionGroup>

## Incorrect liability or revenue figures

| Issue | Check |
| --- | --- |
| Opening balances wrong | [Existing program balances](/user-guide/program-setup/existing-program-balances) |
| Accounting settings changed | Settings apply retrospectively — review [Accounting configuration](/user-guide/program-setup/accounting-configuration) |
| Missing breakage data | Ensure Breakage Levels uploaded for each period |
| Partner code mismatch | Verify codes match exactly between Partners and CSV uploads |

## Partner not appearing in dropdown

- Confirm partner status is **Active**
- Refresh the page after saving partner configuration
