The **Growth** section helps you model future program outcomes and run standalone calculations outside your uploaded actuals.

## Available tools

| Menu item | Purpose |
| --- | --- |
| **Projections** | Forward-looking scenario modelling for points issuance, redemption, and financial outcomes |
| **Calculator** | Ad-hoc calculations for a bottom-up program projection |

<Frame caption="Fidivio projections and calculator">
  ![Growth tools](/images/user-guide/platform-navigation/growth-tools.png)
</Frame>

## When to use projections vs. calculator

<AccordionGroup>
  <Accordion title="Projections">
    Use when you need multi-period scenario analysis tied to your configured program settings and uploaded baseline data.
  </Accordion>
  <Accordion title="Calculator">
    Use for quick, standalone calculations — for example, testing a single assumption change without updating uploaded data.
  </Accordion>
</AccordionGroup>

<Tip>
Ensure your program setup and historical data are complete before running projections. Incomplete configuration produces unreliable forward views.
</Tip>
