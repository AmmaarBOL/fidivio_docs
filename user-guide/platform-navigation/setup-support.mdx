**Setup & Support** is where administrators configure the platform, manage partners and data, and access documentation and support.

## Menu overview

| Menu item | Sub-sections | Purpose |
| --- | --- | --- |
| **Partners** | — | Manage partner organisations that issue or redeem points |
| **Data Input** | Multiple sub-sections | Manual entry or bulk CSV upload of program data |
| **Configuration** | Program Setup, User Management, Role Management, Feature Management, Connectors | Platform and program configuration |
| **Help & Support** | Multiple sub-sections | Documentation, support contact, and tickets |
