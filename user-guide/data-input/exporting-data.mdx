---
title: "Exporting data"
description: "Export program data as CSV from any data category."
---

All data can be exported as CSV from each data category page.

<Steps>
  <Step title="Open the data category">
    Navigate to the relevant category under **Data Input**.
  </Step>
  <Step title="Click Export">
    Click **Export** in the top right corner.
  </Step>
  <Step title="Select date range">
    Choose the desired data range in the on-screen form.
  </Step>
  <Step title="Download">
    Click **Export** again to download the file.
  </Step>
</Steps>

## Common use cases

- Obtain record `id` values for update or delete operations in batch uploads
- Audit uploaded data against source systems
- Share data extracts with finance or audit teams

<Note>
Exported files include the `id` column needed for `update` and `delete` actions in subsequent batch uploads.
</Note>
