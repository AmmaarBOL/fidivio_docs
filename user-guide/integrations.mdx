## Setting Up an Integration

<Steps>
  <Step title="Navigate to Integrations">
    Go to **Settings \> Integrations**.
  </Step>
  <Step title="Browse Connectors">
    Find the platform you want to connect from the integrations catalog.
  </Step>
  <Step title="Configure Credentials">
    Enter your API keys or OAuth credentials for the target platform.
  </Step>
  <Step title="Map Data Fields">
    Map fields between Fidivio and the connected system (e.g., email, order ID).
  </Step>
  <Step title="Test the Connection">
    Run a test transaction to verify the integration is working correctly.
  </Step>
  <Step title="Go Live">
    Enable the integration for production traffic.
  </Step>
</Steps>

## Webhooks

Webhooks allow Fidivio to push real-time event notifications to your systems.

### Available Webhook Events

| Event | Description |
| --- | --- |
| `member.enrolled` | New member registered |
| `member.tier_changed` | Member tier promoted or demoted |
| `transaction.earned` | Points earned on a transaction |
| `transaction.redeemed` | Points redeemed for a reward |
| `transaction.reversed` | Transaction reversed |
| `points.expiring` | Points approaching expiry date |
| `campaign.reward_issued` | Campaign reward awarded to member |

### Configuring Webhooks

1. Go to **Settings \> Integrations \> Webhooks**
2. Click **\+ Add Webhook**
3. Enter your endpoint URL (must be HTTPS)
4. Select events to subscribe to
5. Set a secret key for payload verification
6. Save and test the webhook

### Webhook Payload Example

```json
{
  "event": "transaction.earned",
  "timestamp": "2026-05-24T10:30:00Z",
  "data": {
    "transaction_id": "txn_abc123",
    "member_id": "mem_xyz789",
    "points_earned": 150,
    "transaction_value": 75.00,
    "currency": "USD",
    "channel": "pos",
    "location_code": "STORE_001"
  }
}
```

### Webhook Security

Verify webhook authenticity using the HMAC signature in the `X-Fidivio-Signature` header:

```python
import hmac
import hashlib

def verify_webhook(payload: bytes, signature: str, secret: str) -> bool:
    expected = hmac.new(
        secret.encode(),
        payload,
        hashlib.sha256
    ).hexdigest()
    return hmac.compare_digest(f"sha256={expected}", signature)
```

## API Authentication

Fidivio APIs use Bearer token authentication:

```bash
curl -X GET https://api.fidivio.com/v1/members \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

Generate API keys in **Settings \> API Keys \> \+ Generate Key**.

<Note>
  API keys have configurable scopes (read-only, read-write, admin). Follow the principle of least privilege when generating keys for integrations.
</Note>
