# Settings

The Settings section controls platform-level configuration for your Fidivio organization.

## Organization Settings

Go to **Settings \> Organization** to configure:

| Setting | Description |
| --- | --- |
| **Organization Name** | Your brand name shown in the platform and emails |
| **Logo** | Upload light and dark mode logos (PNG, SVG) |
| **Primary Color** | Brand color for member-facing interfaces |
| **Base Currency** | Default currency for transaction processing |
| **Default Language** | Platform interface language |
| **Supported Languages** | Languages available in the member portal |
| **Time Zone** | Used for scheduling, reporting, and timestamps |
| **Fiscal Year Start** | Month your fiscal year begins (for annual reporting) |

## User Management

### Creating Admin Users

1. Go to **Settings \> Users \> \+ Invite User**
2. Enter name and email address
3. Assign a role:

| Role | Capabilities |
| --- | --- |
| **Super Admin** | Full access to all modules and settings |
| **Program Manager** | Manage programs, campaigns, and rules |
| **Customer Service** | View/edit member profiles and transactions |
| **Analyst** | Read-only access to analytics and reports |
| **Finance** | Access to financial reports and liability data |
| **Custom** | Configurable permissions per module |

4. Click **Send Invitation**
5. User receives an email invitation to set up their account

### Managing Permissions

For custom roles, configure granular permissions per module:

1. Go to **Settings \> Roles \> \+ Create Role**
2. Name the role
3. Toggle permissions for each module:
   - **View**: Can see data
   - **Create**: Can create new records
   - **Edit**: Can modify existing records
   - **Delete**: Can delete records
   - **Export**: Can export data
4. Save and assign to users

## Security Settings

### Multi-Factor Authentication

- Go to **Settings \> Security \> MFA**
- Enable MFA requirement for all users or specific roles
- Supported methods: Authenticator app (TOTP), SMS, Email OTP

### Session Management

- Configure session timeout duration
- Set maximum concurrent sessions per user
- View and revoke active sessions

### IP Allowlisting

- Restrict platform access to specific IP ranges
- Useful for limiting admin access to corporate network

### Audit Log

- All administrative actions are recorded in **Settings \> Audit Log**
- Filter by user, action type, date range
- Export audit log for compliance reporting

## Notification Templates

Customize the emails and SMS messages sent to members:

1. Go to **Settings \> Notifications**
2. Select a template:
   - Welcome / Enrollment Confirmation
   - Points Earned
   - Points Redeemed
   - Tier Upgrade
   - Tier Downgrade Warning
   - Points Expiry Alert
   - Password Reset
3. Edit the template using the HTML/text editor
4. Use template variables: `{{member.first_name}}`, `{{points.balance}}`, etc.
5. Send a test email to preview
6. Save and activate

## Data Management

### Data Retention

Configure how long different data types are retained:

- Transaction records: Default 7 years
- Member profiles: Until account deletion request
- Audit logs: Default 3 years

### Data Export

Schedule automated data exports to an S3 bucket or SFTP server:

1. Go to **Settings \> Integrations \> Data Exports**
2. Configure destination (S3 bucket, SFTP)
3. Select data types and format (CSV, JSON, Parquet)
4. Set export frequency (daily, weekly)
5. Activate the export

### GDPR / Privacy

- Process right-to-erasure requests: **Members \> \[Member\] \> Actions \> Delete Data**
- Export member data for portability requests: **Members \> \[Member\] \> Actions \> Export Data**
- Configure consent management in **Settings \> Privacy**
