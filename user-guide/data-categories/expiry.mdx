---
title: "Expiry"
description: "Record points formally expired and removed from member accounts."
---

Expiry records capture actual points voided and removed from member accounts in each period.

**Location:** Setup & Support → Data Input → Expiry

## When to use

Enter an expiry record in any period where the program formally expires points. No entry is needed if no points expired.

Used when Breakage Recognition is set to Setting 3 (actual expiry events) and for points movement tracking on the Points Overview Dashboard.

## Fields

| Field | Definition | Format | Required |
| --- | --- | --- | --- |
| `points_expired` | Total points formally expired and removed on this transaction date | Numeric ≥ 0 | Yes |
