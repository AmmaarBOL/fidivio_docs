---
title: "Program costs"
description: "Upload operational costs of running the loyalty program."
---

Program Costs records running costs excluding direct redemption fulfilment costs (captured under Redemptions). Examples include staff, technology licences, office expenses, and marketing spend.

**Location:** Setup & Support → Data Input → Program Costs

<Note>
Program Costs captures operational overheads only. Direct redemption fulfilment costs are entered under Redemptions.
</Note>

## Pre-configuration required

`cost_category` and `cost_item` must be set up before uploading — same process as Other Revenue Sources.

<Warning>
Category and item values must be created first — they cannot be entered during batch data upload.
</Warning>

## Fields

| Field | Definition | Format | Required |
| --- | --- | --- | --- |
| `cost_category` | High-level category (e.g. Staff Costs, Technology) | Text | Optional |
| `cost_item` | Specific item (e.g. Platform Licence Fee) | Text | Optional |
| `currency` | Currency of the cost | ISO 4217 | Optional |
| `total_running_costs` | Total cost for this item in this period | Numeric ≥ 0 | Optional |
