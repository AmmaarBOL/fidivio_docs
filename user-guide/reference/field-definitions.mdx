This page consolidates field definitions for all data categories. Common fields appear in every template.

## Common fields (all categories)

| Field | Definition | Format | Required |
| --- | --- | --- | --- |
| `action` | Operation: insert, update, or delete | `insert` \| `update` \| `delete` | Yes |
| `id` | Unique record identifier. Required for update/delete | Integer | Conditional |
| `transaction_date` | Date associated with the record | `yyyy-mm-dd` | Yes |
| `external_unique_identifier` | Optional cross-system reference ID | Integer | Optional |
