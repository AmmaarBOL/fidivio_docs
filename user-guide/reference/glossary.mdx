---
title: "Glossary"
description: "Key terms used in Fidivio documentation."
---

| Term | Definition |
| --- | --- |
| **Accrual** | Points issued to members in a period, with associated yield |
| **Applied SSP** | Standalone selling price used in model calculations, which may be adjusted for breakage |
| **Breakage** | Portion of points not expected to be redeemed |
| **Breakage income** | Revenue recognized from expected or actual point breakage |
| **Contract liability** | Deferred revenue obligation for outstanding points (IFRS 15) |
| **Conventional model** | Program run as cost center with revenue deferral only |
| **Deferred revenue** | Revenue deferred until performance obligations are satisfied |
| **Performance obligation** | Promise to transfer goods or services — loyalty points represent a PO |
| **Profit centre model** | Program run as P&L with points sold to partners |
| **Redemption** | Points exchanged by members for rewards |
| **SSP** | Standalone Selling Price — value of one point if sold separately |
| **True-up** | Catch-up adjustment to restate contract liability when estimates change |
| **Yield** | Revenue earned per point issued (profit-centre programs) |

<Tip>
For detailed accounting definitions, see [Accounting definitions](/technical-memo/accounting-definitions) in the Technical Memorandum.
</Tip>
