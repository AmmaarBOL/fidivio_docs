<AccordionGroup>
  <Accordion title="Can I change accounting settings after go-live?">
    Yes, but the latest configuration is applied retrospectively to all historical data. Establish settings at program set-up and change only with finance team approval. See [Accounting configuration](/user-guide/program-setup/accounting-configuration).
  </Accordion>
  <Accordion title="Why did my CSV upload fail?">
    Common causes: unknown partner codes, wrong date format, missing pre-configured categories, or invalid required fields. Download the handback file from the Batches tab. See [Upload errors](/user-guide/troubleshooting/upload-errors).
  </Accordion>
  <Accordion title="Do I need Sales Information data?">
    Only if your Program Type is **Conventional Model**. Profit-centre programs do not require Sales Information. See [Sales information](/user-guide/data-categories/sales-information).
  </Accordion>
  <Accordion title="What is the difference between IFRS P&L and Adjusted EBITDA P&L?">
    IFRS P&L follows revenue recognition rules; Adjusted EBITDA P&L presents a management view with gross billings and specific adjustments. Fidivio reconciles the two. See [Financial dashboards](/user-guide/platform-navigation/financial-dashboards).
  </Accordion>
  <Accordion title="How do I get record IDs for updates?">
    Export data from the relevant category using **Export**. The CSV includes the `id` column. See [Exporting data](/user-guide/data-input/exporting-data).
  </Accordion>
  <Accordion title="Can I upload data for inactive partners?">
    No. CSV uploads referencing Inactive partner codes are rejected. Set partners to Active before uploading.
  </Accordion>
  <Accordion title="Where can I find detailed calculation formulas?">
    See the [Technical Memorandum](/technical-memo/introduction) for model inputs, configuration parameters, and calculation detail.
  </Accordion>
</AccordionGroup>
