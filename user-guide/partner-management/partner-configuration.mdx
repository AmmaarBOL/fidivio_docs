title: "Partner configuration"
description: "Field reference for configuring a partner in Fidivio."
---

Complete these fields when adding or editing a partner. Partner codes must match exactly in all CSV imports — they are case-sensitive.

| Field | Description | Required |
| --- | --- | --- |
| **Partner Name** | Full legal or trading name of the partner organisation | Yes |
| **Partner Code** | Short unique identifier for data uploads (e.g. AIRX, BANK01). Must match exactly in all CSV imports | Yes |
| **Category** | Partner category (e.g. Finance, Airline, Retail). Type a new name and click **Create New** to add | Yes |
| **Partner Location** | Geographic location of the partner | Optional |
| **Partner Status** | **Active** or **Inactive** | Yes |
| **Affiliation** | **Internal** (program operator's company) or **External** (third party) | Yes |
| **Type** | **Accrual**, **Redemption**, or **Both** | Yes |
| **Start Date** | Commencement date of the partnership | Optional |
| **End Date** | End date of the partnership. Typically populated for Inactive partners | Optional |
| **Program / Company Logo** | Partner logo displayed on the Partner Dashboard | Optional |

<Note>
Partner codes must be set up under **Setup & Support → Partners** before accrual or redemption data can be uploaded.
</Note>
