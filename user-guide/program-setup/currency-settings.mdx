---
title: "Currency settings"
description: "Configure how monetary values and dates display across Fidivio."
---

Currency settings control formatting and precision for all calculations and dashboard displays.

Navigate to **Setup & Support → Configuration → Program Setup → Currency Settings**.

## Fields

| Field | Description | Format | Required |
| --- | --- | --- | --- |
| **Currency Format** | Number formatting for monetary values (e.g. $1,234.56) | Selection | Yes |
| **Date Format** | Date display format (e.g. DD/MM/YYYY) | Selection | Yes |
| **Decimal Precision for Calculation** | Decimal places used in backend calculations | Numeric | Yes |
| **Decimal Precision for Display** | Decimal places shown in dashboards and reports | Numeric | Yes |

<Tip>
Set calculation precision equal to or higher than display precision. Lowering calculation precision can cause rounding differences between detailed and summary views.
</Tip>
