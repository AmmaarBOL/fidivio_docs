# Member Management

Fidivio provides comprehensive tools to manage your loyalty program members throughout their lifecycle — from enrollment to ongoing engagement and offboarding.

## Member Profile

Every member has a unified profile that consolidates:

  <Card title="Program Membership" icon="star">

  <Card title="Transaction History" icon="receipt">

  </Card>

  <Card title="Communication History" icon="envelope">

## Enrolling New Members

### Manual Enrollment

1. Go to **Members \> All Members \> \+ Enroll Member**
2. Enter member details:
   - First Name, Last Name (required)
   - Email Address (required, must be unique)
   - Phone Number
   - Date of Birth
   - Preferred Language
3. Select the program(s) to enroll the member in
4. Click **Enroll**

### Bulk Import

To import multiple members at once:

1. Go to **Members \> Import**
2. Download the **CSV Template**
3. Fill in member data following the template format
4. Upload the completed CSV file
5. Review the validation summary
6. Confirm the import

**Required CSV columns:**

| Column | Required | Format |
| --- | --- | --- |
| `first_name` | Yes | Text |
| `last_name` | Yes | Text |
| `email` | Yes | Valid email |
| `phone` | No | E.164 format |
| `date_of_birth` | No | YYYY-MM-DD |
| `program_code` | Yes | Program identifier |
| `enrollment_date` | No | YYYY-MM-DD (defaults to today) |
| `opening_points` | No | Integer |

<Warning>
  Duplicate email addresses will be flagged during validation. Fidivio will not create duplicate member accounts.
</Warning>

## Searching for Members

Use the search bar in **Members \> All Members** to search by:

- Email address
- Phone number
- Membership ID
- First or last name

Apply filters for:

- Program enrollment
- Tier status
- Points balance range
- Enrollment date range
- Activity status (Active / Inactive / Churned)

## Member Profile Actions

From a member’s profile, administrators can:

<AccordionGroup>
  <Accordion title="Edit Profile">
    Update personal information, contact details, and communication preferences.
  </Accordion>

  <Accordion title="Adjust Points">
    Manually credit or debit points from the member’s balance. Requires a reason code.
  </Accordion>

  <Accordion title="Change Tier">
    Override the system-calculated tier and manually assign a tier. Useful for VIP onboarding.
  </Accordion>

  <Accordion title="Merge Accounts">
    Merge duplicate member accounts. Points and history from both accounts are consolidated.
  </Accordion>

  <Accordion title="Suspend Member">
    Temporarily suspend a member account, preventing earning and redemption activity.
  </Accordion>

  <Accordion title="Deactivate Member">
    Permanently close a member account. This action is irreversible.
  </Accordion>

  <Accordion title="Send Communication">
    Trigger a manual email or SMS to the member from a configured template.
  </Accordion>
</AccordionGroup>

## Member Segments

Segments allow you to group members based on shared attributes for targeted campaigns and reporting.

### Creating a Segment

1. Go to **Members \> Segments \> \+ New Segment**
2. Enter segment name and description
3. Define segment criteria using the rule builder:
   - **Demographics**: Age, location, language
   - **Program Status**: Tier, enrollment date, program
   - **Behavior**: Transaction frequency, recency, average spend
   - **Points**: Balance, points earned/redeemed, expiry status
4. Preview the estimated member count
5. Save the segment

### Segment Types

| Type | Description |
| --- | --- |
| **Static** | Fixed list of members at creation time |
| **Dynamic** | Auto-updates as members meet/leave criteria |

<Tip>
  Use dynamic segments for campaigns targeting "members at risk of churning" or "members close to tier upgrade" to keep your audiences current without manual maintenance.
</Tip>

## Member Activity Feed

The activity feed on a member’s profile shows a chronological history of all:

- Earning transactions
- Redemption transactions
- Point adjustments
- Tier changes
- Campaign participations
- Communication events

Filter the activity feed by activity type and date range for efficient investigation.
