Follow this path to go from first login to populated dashboards. Each step links to detailed documentation.

<Steps>
  <Step title="Log in and set up 2FA">
    Access the platform and complete two-factor authentication on first sign-in.
    → [First login](/user-guide/getting-started/first-login)
  </Step>
  <Step title="Configure your program">
    Set program info, currency settings, and accounting rules.
    → [Program information](/user-guide/program-setup/program-information)
  </Step>
  <Step title="Add partners">
    Register partner organisations before uploading any data.
    → [Managing partners](/user-guide/partner-management/managing-partners)
  </Step>
  <Step title="Set up users and roles">
    Invite your team and assign appropriate permissions.
    → [Roles & permissions](/user-guide/user-role-management/roles-permissions)
  </Step>
  <Step title="Upload data and review dashboards">
    Import historical or current data, then validate on dashboards.
    → [Data input overview](/user-guide/data-input/overview)
  </Step>
</Steps>

## Before you begin

You'll need:

- Admin access to Fidivio
- Your program's fiscal year start month and currency
- Partner codes and historical data (if migrating an existing program)
- Accounting treatment decisions (SSP, breakage, true-up settings)

<Warning>
Complete program setup and partner configuration before uploading data. Uploads referencing unknown partner codes will fail.
</Warning>
