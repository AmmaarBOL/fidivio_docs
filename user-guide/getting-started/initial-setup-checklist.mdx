---
title: "Initial setup checklist"
description: "Complete these admin tasks before using Fidivio for reporting."
---

Complete these steps in order before relying on dashboards and reports. Skipping steps may cause upload failures or incorrect calculations.

<Steps>
  <Step title="Set up the program">
    **Setup & Support → Configuration → Program Setup**

    Configure program info, currency settings, opening balances (if migrating), and accounting rules.
    → [Program information](/user-guide/program-setup/program-information)
  </Step>
  <Step title="Set up partners">
    **Setup & Support → Partners**

    Add all accrual and redemption partners with correct partner codes.
    → [Managing partners](/user-guide/partner-management/managing-partners)
  </Step>
  <Step title="Set up users and roles">
    **Setup & Support → Configuration → User Management** and **Role Management**

    Create roles, invite users, and assign permissions.
    → [Creating roles](/user-guide/user-role-management/creating-roles)
  </Step>
  <Step title="Input program data">
    **Setup & Support → Data Input**

    Upload historical or current data via manual entry or CSV batch upload.
    → [Data input overview](/user-guide/data-input/overview)
  </Step>
  <Step title="Review dashboards">
    Navigate to **Dashboards** and confirm key metrics are populating correctly.
    → [Dashboard overview](/user-guide/platform-navigation/dashboard-overview)
  </Step>
</Steps>

## Checklist summary

| Step | Location | Status |
| --- | --- | --- |
| Program setup | Configuration → Program Setup | ☐ |
| Partners configured | Partners | ☐ |
| Roles created | Configuration → Role Management | ☐ |
| Users invited | Configuration → User Management | ☐ |
| Data uploaded | Data Input | ☐ |
| Dashboards reviewed | Dashboards → Home | ☐ |

<Warning>
Accounting settings should be established at program set-up and applied consistently. Changing settings mid-program applies the latest configuration retrospectively to all historical data.
</Warning>

<Tip>
Establish partner code naming conventions before your first upload. Codes are case-sensitive and must match exactly in all CSV imports.
</Tip>
