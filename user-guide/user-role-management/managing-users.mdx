title: "Managing users"
description: "Add, edit, and deactivate users in Fidivio."
---

User Management allows administrators to add platform users, update details, and manage account security.

**Location:** Setup & Support → Configuration → User Management

## Add a new user

<Steps>
  <Step title="Open User Management">
    Navigate to **Setup & Support → Configuration → User Management**.
  </Step>
  <Step title="Click Add User">
    Click **+ Add User** in the top right corner.
  </Step>
  <Step title="Complete fields">
    Enter **First Name**, **Last Name**, **Email Address**, and **Role**. Email is the login credential.
  </Step>
  <Step title="Confirm">
    Click **Add User**. The user receives an email invitation to set up password and 2FA.
  </Step>
</Steps>

<Info>
The invitation email is sent from a Fidivio system address. Ask new users to check spam or junk folders if the email does not arrive within a few minutes.
</Info>

## Edit user details

1. Click the **edit (pen) icon** on the user's row.
2. Update name, email, or assigned role.
3. Click **Update Details**.

<Note>
Changing a user's role takes effect immediately. Permissions update on their next page load or login.
</Note>

## Deactivate a user

1. Click the **three-dots icon** on the user's row.
2. Select **Deactivate Account** and confirm.

Deactivated users cannot log in. Historical activity and data contributions are retained. Accounts can be reactivated by an administrator.
