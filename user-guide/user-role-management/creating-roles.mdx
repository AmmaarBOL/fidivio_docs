## Create a new role

<Steps>
  <Step title="Open Role Management">
    Navigate to **Setup & Support → Configuration → Role Management**.
  </Step>
  <Step title="Click Create Role">
    Click **+ Create Role** in the top right corner.
  </Step>
  <Step title="Name the role">
    Enter a **Role Title** that describes the user group (e.g. Finance Analyst, Read-Only Auditor, Program Manager).
  </Step>
  <Step title="Select permissions">
    Review available activities and check boxes for each permitted action. Activities are grouped by module.
  </Step>
  <Step title="Save">
    Click **Create Role**. The role appears in the list and is available for user assignment.
  </Step>
</Steps>

## Edit a role

1. Click the **edit (pen) icon** on the role row.
2. Modify the Role Title or adjust activity permissions.
3. Click **Save**. All assigned users inherit updated permissions immediately.

<Tip>
Create a read-only auditor role with view-only report access before granting full admin permissions to new team members.
</Tip>
