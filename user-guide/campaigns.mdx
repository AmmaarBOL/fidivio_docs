# Campaigns & Promotions

Campaigns in Fidivio allow you to run time-limited, targeted promotions to drive member engagement, reward loyal behavior, and re-activate dormant members.

## Campaign Types

<CardGroup cols={2}>
  <Card title="Bonus Points" icon="coins">
    Award bonus points to qualifying members during a promotional period (e.g., double points weekend).
  </Card>

  <Card title="Bonus Transactions" icon="bolt">
    Trigger a reward when a member completes a specific number of transactions.
  </Card>

  <Card title="Spend Threshold" icon="chart-line">
    Award bonus points or a reward when a member reaches a spend milestone.
  </Card>

  <Card title="Re-engagement" icon="arrow-rotate-left">
    Target inactive members with a win-back offer to bring them back.
  </Card>

  <Card title="Tier Accelerator" icon="rocket">
    Temporarily boost tier qualification progress to incentivize upgrades.
  </Card>

  <Card title="Birthday/Anniversary" icon="cake-candles">
    Automatically reward members on their birthday or anniversary date.
  </Card>
</CardGroup>

## Creating a Campaign

<Steps>
  <Step title="Navigate to Campaigns">
    Go to **Campaigns \> All Campaigns \> \+ Create Campaign**.
  </Step>
  <Step title="Select Campaign Type">
    Choose the type of campaign that matches your objective.
  </Step>
  <Step title="Basic Details">
    Enter:

    - Campaign name and description
    - Start and end dates/times
    - Campaign budget (optional: maximum total points to award)
  </Step>
  <Step title="Target Audience">
    Define who qualifies:

    - **All Members**: No restrictions
    - **Specific Segment**: Select a pre-built member segment
    - **Tier-Specific**: Only members at selected tiers qualify
    - **Custom Rules**: Build custom eligibility criteria
  </Step>
  <Step title="Configure Reward">
    Define the reward:

    - Bonus points amount or multiplier
    - Voucher/discount value
    - Specific product reward
  </Step>
  <Step title="Trigger Conditions">
    Set what action triggers the reward:

    - Any qualifying transaction
    - Transaction value ≥ \$X
    - Specific product/category purchase
    - N-th transaction milestone
  </Step>
  <Step title="Frequency Cap">
    Optionally limit how often a member can earn the campaign reward:

    - Once per campaign
    - N times per day/week/month
    - Once per qualifying transaction
  </Step>
  <Step title="Notifications">
    Configure member communications:

    - Campaign launch email/SMS
    - Mid-campaign reminder
    - Reward earned confirmation
  </Step>
  <Step title="Review & Launch">
    Review all settings in the summary screen and click **Launch Campaign**.
  </Step>
</Steps>

## Campaign Status

| Status | Description |
| --- | --- |
| **Draft** | Being configured, not yet active |
| **Scheduled** | Approved and waiting for start date |
| **Active** | Currently running |
| **Paused** | Temporarily suspended |
| **Ended** | Past end date, no longer awarding rewards |
| **Cancelled** | Manually stopped before completion |

## Campaign Performance

Monitor your campaigns in real time:

1. Go to **Campaigns \> All Campaigns \> \[Campaign Name\]**
2. View the **Performance** tab for:

| Metric | Description |
| --- | --- |
| **Participants** | Unique members who triggered the campaign |
| **Total Rewards Issued** | Total points/rewards awarded |
| **Conversion Rate** | % of eligible members who participated |
| **Incremental Revenue** | Estimated revenue driven by the campaign |
| **Cost per Participant** | Average rewards cost per participating member |

## Campaign Templates

Save time by creating reusable campaign templates:

1. After building a campaign, click **Save as Template**
2. Give the template a name
3. Templates are available under **Campaigns \> Templates**
4. When creating a future campaign, select **Start from Template**

<Tip>
  Build templates for your recurring campaigns (e.g., monthly double-points weekend, seasonal promotions) and simply adjust the dates each time.
</Tip>
