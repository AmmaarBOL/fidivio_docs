# Analytics & Reports

Fidivio provides a comprehensive analytics suite to help you measure program effectiveness, understand member behavior, and optimize your loyalty strategy.

## Analytics Overview

Access analytics at **Analytics \> Overview** for a high-level summary of your program health:

<CardGroup cols={2}>
  <Card title="Program Performance" icon="chart-line">
    Points issued, redeemed, expired, and overall redemption rate trends
  </Card>

  <Card title="Member Growth" icon="users">
    New enrollments, active members, churn rate, and retention metrics
  </Card>

  <Card title="Revenue Impact" icon="dollar-sign">
    Incremental revenue, average order value lift, and purchase frequency
  </Card>

  <Card title="Campaign Performance" icon="megaphone">
    Campaign participation, conversion rates, and ROI across all campaigns
  </Card>
</CardGroup>

## Standard Reports

### Member Reports

| Report | Description |
| --- | --- |
| **Member Summary** | Total members by tier, status, and enrollment period |
| **Member Activity** | Active vs. inactive member breakdown |
| **Enrollment Trend** | New member registrations over time |
| **Tier Distribution** | Member count and movement across tiers |
| **Churn Analysis** | Members at risk of churning based on inactivity |
| **Lifetime Value** | Member LTV segmentation |

### Points Reports

| Report | Description |
| --- | --- |
| **Points Issuance** | Total points issued by period, channel, and program |
| **Points Redemption** | Total points redeemed by reward type and channel |
| **Redemption Rate** | Redemption rate trends over time |
| **Points Liability** | Outstanding points balance (program financial liability) |
| **Points Expiry** | Points expiring in the next 30/60/90 days |

### Financial Reports

| Report | Description |
| --- | --- |
| **Loyalty Liability** | Current outstanding points value in currency |
| **Breakage Rate** | Percentage of points that expire unredeemed |
| **Cost per Point** | Average cost to issue and honor one point |
| **Campaign ROI** | Revenue vs. cost for each campaign |

## Custom Reports

Build tailored reports using the custom report builder:

<Steps>
  <Step title="Open Report Builder">
    Go to **Analytics \> Custom Reports \> \+ New Report**.
  </Step>
  <Step title="Select Data Source">
    Choose from: Members, Transactions, Campaigns, Points, Tiers.
  </Step>
  <Step title="Select Metrics">
    Choose the metrics and dimensions to include in your report.
  </Step>
  <Step title="Apply Filters">
    Filter by date range, program, tier, segment, channel, or location.
  </Step>
  <Step title="Choose Visualization">
    Select: Table, Bar Chart, Line Chart, Pie Chart, or Heatmap.
  </Step>
  <Step title="Save & Schedule">
    Save the report and optionally schedule automated delivery to email recipients.
  </Step>
</Steps>

## Dashboards

Create custom analytics dashboards by combining multiple reports and charts:

1. Go to **Analytics \> Dashboards \> \+ New Dashboard**
2. Give the dashboard a name
3. Click **\+ Add Widget** to add reports
4. Arrange and resize widgets
5. Share with team members or set as default view

## Exporting Reports

All reports can be exported as:

- **CSV**: Raw data for Excel/BI tools
- **PDF**: Formatted report for sharing
- **PNG/SVG**: Chart images for presentations

## Scheduled Report Delivery

Automate report delivery to stakeholders:

1. Open any saved report
2. Click **Schedule**
3. Set frequency: Daily, Weekly, Monthly
4. Enter recipient email addresses
5. Choose file format
6. Save schedule

<Tip>
  Create a weekly program health report and schedule it to senior stakeholders every Monday morning. Include the Points Liability, Active Members, and Campaign Performance reports on a single dashboard.
</Tip>
