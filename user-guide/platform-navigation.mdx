# Platform Navigation

The left-hand sidebar is your primary navigation. It is divided into three sections: **Dashboards**, **Growth**, and **Setup & Support**.

---

## 2.1 Dashboards

| Menu Item | Sub-sections | Purpose |
|---|---|---|
| **Home** | — | High-level overview of all key financial metrics across Accounting, Financial Statements, and Management Reports |
| **Accounting** | IFRS Dashboard | Detailed financial breakdown behind the IFRS P&L |
| | Deferred Revenue Dashboard | Detailed breakdown of deferred revenue movements |
| | Points Movement Dashboard | Overview of points earned, redeemed, expired, and adjusted during the selected period |
| **Financial Statements** | Income Statement | Detailed breakdown for IFRS P&L, Adjusted EBITDA P&L, and reconciliation of the two |
| **Management Reports** | KPI Dashboard | Key operational KPIs for the program |
| | A-EBITDA Waterfall | Detailed financial breakdown behind the Adjusted EBITDA P&L |
| | Partner Dashboard | Detailed points and financial KPIs by partner |
| | Sankey Chart | Visual summary of Gross Billings flow to other key financial components |
| | Partner Margin | Detailed points status, redemption composition and points margin by partner |

## 2.2 Growth

| Menu Item | Sub-sections | Purpose |
|---|---|---|
| **Projections** | — | Forward-looking scenario modelling for points issuance, redemption, and financial outcomes |
| **Calculator** | — | Ad-hoc calculations for a bottom-up program projection |

## 2.3 Setup & Support

| Menu Item | Sub-sections | Purpose |
|---|---|---|
| **Partners** | — | Manage partner organisations that issue or redeem points within your program |
| **Data Input** | Multiple sub-sections | Manual input or bulk upload of transaction data for the program |
| **Configuration** | Program Setup | Configure the key attributes of your program and its financial parameters |
| | User Management | Manage or create new users for different roles |
| | Role Management | Create and manage roles and their access permissions |
| | Feature Management | Enable or disable platform features |
| | Connectors | Configure integrations with external data sources |
| **Help & Support** | Multiple sub-sections | Access documentation, contact support, and submit tickets |
