# Dashboard Overview

The Fidivio dashboard is your central command center for monitoring program performance, member activity, and business metrics in real time.

## Dashboard Layout

The dashboard is organized into four main areas:

```text
┌─────────────────────────────────────────────────────┐
│  Top Navigation Bar                                  │
├──────────┬──────────────────────────────────────────┤
│          │  Main Content Area                        │
│  Left    │  ┌──────────┐  ┌──────────┐             │
│  Sidebar │  │ KPI Card │  │ KPI Card │  ...        │
│          │  └──────────┘  └──────────┘             │
│          │  ┌────────────────────────────┐          │
│          │  │ Charts & Analytics         │          │
│          │  └────────────────────────────┘          │
└──────────┴──────────────────────────────────────────┘
```

## Key Performance Indicators (KPIs)

The top of the dashboard displays real-time KPI cards:

| KPI | Description |
| --- | --- |
| **Active Members** | Total members with activity in the last 30 days |
| **Points Issued** | Total points issued in the current period |
| **Points Redeemed** | Total points redeemed in the current period |
| **Redemption Rate** | Percentage of issued points that have been redeemed |
| **New Enrollments** | New member registrations in the current period |
| **Program Revenue** | Revenue attributed to loyalty program activity |

## Date Range Selector

All dashboard data can be filtered by date range using the selector in the top-right corner:

- Today
- Last 7 Days
- Last 30 Days
- Last Quarter
- Last Year
- Custom Range

## Customizing Your Dashboard

You can customize dashboard widgets by:

1. Clicking the **Customize** button (grid icon) in the top right
2. Drag and drop widgets to reorder them
3. Click the toggle on any widget to show/hide it
4. Click **Save Layout** to persist your changes

<Tip>
  Pin your most important KPIs to the top of the dashboard for quick access. Use the star icon on any metric card to pin it.
</Tip>
