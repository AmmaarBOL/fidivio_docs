Before inputting data, select settings for your program type and preferred accounting treatment. Multiple approaches are permitted under current accounting rules.

## Four parameters

<CardGroup cols={2}>
  <Card title="Parameter 1: Program type" icon="building" href="/technical-memo/configuration/program-type">
    Profit center vs conventional model.
  </Card>
  <Card title="Parameter 2: Liability true-up" icon="scale" href="/technical-memo/configuration/liability-true-up">
    Impact of breakage and SSP estimate changes.
  </Card>
  <Card title="Parameter 3: Breakage protocol" icon="clock" href="/technical-memo/configuration/breakage-protocol">
    How breakage income is recognized each period.
  </Card>
  <Card title="Parameter 4: Applied SSP" icon="tag" href="/technical-memo/configuration/applied-ssp">
    Whether SSP is adjusted for expected breakage.
  </Card>
</CardGroup>

The key parameters include:

- The type of program (conventional or profit center)
- The impact of changes to breakage and standalone selling prices
- The protocol based on which the breakage income is recognized
- The type of standalone selling price (adjusted for breakage or not)

<Warning>
These parameter settings in the configuration menu can only be set-up at the start. Once the program gets underway, the parameters cannot be changed.
</Warning>
